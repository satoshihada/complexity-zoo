Class Description

SO[ t ( n ) {\displaystyle t(n)} ]: Iterated Second-Order logic

SO[ t ( n ) {\displaystyle t(n)} ] is to SO as FO[ t ( n ) {\displaystyle t(n)} ] is to FO. But we now also have second-order quantifier in the quantifier block.

In Descriptive complexity we can see that :

  • SO[ n O ( 1 ) {\displaystyle n^{O(1)}} ] is equal to PSPACE it is also another way to write SO(TC)
  • SO[ 2 n O ( 1 ) {\displaystyle 2^{n^{O(1)}}} ] is equal to EXPTIME it is also another way to write SO(LFP)
