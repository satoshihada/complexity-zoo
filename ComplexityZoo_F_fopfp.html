FO(PFP): First-order with partial fixed point

FO(pfp) is the set of boolean queries definable with first-order formulae with a partial fixed point operator.

Let k {\displaystyle k} be an integer, x , y {\displaystyle x,y} vectors of k {\displaystyle k} variables, P {\displaystyle P} a second-order variable of arity k {\displaystyle k} , and ϕ {\displaystyle \phi } a FO(PFP) function using x {\displaystyle x} and P {\displaystyle P} as variables, then we can define iteratively ( P i ) i ∈ N {\displaystyle (P_{i})_{i\in N}} such that P 0 ( x ) = f a l s e {\displaystyle P_{0}(x)=false} and P i ( x ) = ϕ ( P i − 1 , x ) {\displaystyle P_{i}(x)=\phi (P_{i-1},x)} which means that the property P i {\displaystyle P_{i}} is true on the input x {\displaystyle x} if ϕ {\displaystyle \phi } is true on input x {\displaystyle x} , and when the variable P {\displaystyle P} is replaced by P i − 1 {\displaystyle P_{i-1}} . Then, either there is a fixed point, or the list of ( P i ) {\displaystyle (P_{i})} is looping.

PFP( ϕ P , x ) ( y ) {\displaystyle \phi _{P,x})(y)} is defined as the value of the fixed point of ( P i ) {\displaystyle (P_{i})} on y if there is a fixed point, else as false.

Since P {\displaystyle P} s are property of arity k {\displaystyle k} , there is at most 2 n k {\displaystyle 2^{n^{k}}} values for the P i {\displaystyle P_{i}} s, so with a poly-space counter we can check if there is a loop or not.

It was proved in [Imm98] that FO(pfp) is equal to PSPACE.

FO(LFP): First-order with least fixed point

FO(LFP) is the set of boolean queries definable with first-order fixed-point formulae where the partial fixed point is limited to be monotone, which means that if the second order variable is P {\displaystyle P} , then P i ( x ) {\displaystyle P_{i}(x)} always implies P i + 1 ( x ) {\displaystyle P_{i+1}(x)} .

We can obtain the monotony by restricting the formula ϕ {\displaystyle \phi } to have only positive occurrences of P {\displaystyle P} (i.e. there is an even number of negations before every occurrence of P {\displaystyle P} ). We can also describe LFP( ϕ P , x {\displaystyle \phi _{P,x}} ) as syntactic sugar of PFP( ψ P , x {\displaystyle \psi _{P,x}} ) where ψ ( P , x ) = ϕ ( P , x ) ∨ P ( x ) {\displaystyle \psi (P,x)=\phi (P,x)\vee P(x)} .

Thanks to monotonicity we only add and never remove vectors to the truth table of P {\displaystyle P} , and since there is only n k {\displaystyle n^{k}} possible vectors we always find a fixed point before n k {\displaystyle n^{k}} iterations. Hence it was shown in [Imm82] that FO(LFP)=P. This definition is equivalent to FO( n O ( 1 ) {\displaystyle n^{O(1)}} ).
