Class Description

CL#P: Cluster Sharp-P

The class of #P function problems such that some underlying NP machine M {\displaystyle M} witnessing membership in #P has"clustered" accepting paths. That is:

  • There exists a polynomial p {\displaystyle p} such that each computation path of M {\displaystyle M} on each input x {\displaystyle x} is exactly p ( | x | ) {\displaystyle p(|x|)} bits long.
  • There is a length-respecting total order A {\displaystyle A} having polynomial-time computable adjacency checks on the computation paths of M {\displaystyle M} .
  • The accepting paths of M {\displaystyle M} on any input x {\displaystyle x} are contiguous with respect to A {\displaystyle A} .

Defined in [HHK+05].
